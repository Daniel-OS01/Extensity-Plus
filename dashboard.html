About Extensity-Plus

Version:

Extensity-Plus lets you quickly enable, disable, and manage Chrome extensions from the popup, profiles, and URL-based rules.

Repository: Daniel-OS01/Extensity-Plus

Main inspiration: sergiokas/Extensity

Credits: Maintained by Daniel-OS01 and built on the original Extensity work by Sergio Kas, with additional inspiration from the wider Extensity open-source lineage.

Chrome Web Store publishing will be added later. This build currently ships as a direct repository/unpacked extension workflow.

Quick Guide

  • Popup — Toggle extensions. Click the extension name in Table/Compact mode to enable/disable.
  • Profiles — Save extension sets. Apply from popup or keyboard shortcuts.
  • Always On — Extensions in this profile stay enabled when you toggle all off.
  • Favorites — These extensions always appear at the top of the popup.
  • URL Rules — Auto enable/disable extensions when visiting matching URLs.
  • Groups — Color-code extensions for visual organization.
  • Aliases — Give extensions shorter search names.
